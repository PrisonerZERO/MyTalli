Sync

Free-tier Etsy shop cap (1-shop gate)

Multi-shop platforms now respect the plan tier. UI replaces "Connect another shop" with a "Upgrade to Pro" gradient CTA when a free-tier user is at the 1-shop cap. EtsyConnect endpoint enforces the same rule server-side — refuses the OAuth launch and redirects to /platforms?error=etsy_plan_limit with an explanatory toast. Supersedes the originally-planned IsActive gate — blocking the connection is simpler and more honest than letting them connect many and silently syncing one.
