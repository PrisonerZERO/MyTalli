Suggestion Card — Admin Note Concepts

Three placement options for admin notes on suggestion cards

Concept A — Inline Callout

Purple left-border callout below the description. Clean, distinct from user content. Feels like an official annotation — not a conversation. The "Admin:" label makes authorship clear.
